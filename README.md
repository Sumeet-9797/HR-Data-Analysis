# HR Data Analysis Dashboard | Power BI

Project Overview
This project is an interactive HR Analytics Dashboard developed using Power BI to analyze workforce data of 161 employees.
The dashboard provides insights into headcount distribution, salary structure, gender diversity, qualification impact, leave balance, and hiring trends.

Business Objective

To enable HR leadership to make data-driven decisions related to workforce planning, compensation structure, and diversity analysis.

Key Insights

Total Headcount: 161 Employees
Average Salary: $54.2K
Gender Distribution: 55% Female, 45% Male
Salary Range: $28,900 – $85,000
Highest Paying Role: Product Manager
Workforce Growth observed from 2018–2023
Operational roles have highest headcount
